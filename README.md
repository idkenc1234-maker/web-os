# Theo Web OS Kiosk
My custom macOS setup: Electric blue background + purple hand-drawn computer doodle + auto-fullscreen Yahoo portal!

## How to Use
1. Download the .app or .dmg from Releases (below).
2. Double-click to launch (right-click > Open if Gatekeeper warns).
3. Add to Login Items for auto-start: System Settings > General > Login Items > +.
4. Set wallpapers: Right-click PNGs > Set Desktop Picture.
